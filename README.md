# 🎀 Misa Bot

Misa is a Discord bot created for the **Misa Community**.
She helps manage the server, welcome new members and provide useful commands for the community.

Created on **March 5, 2026**.

---

## ✨ Features

* 🎀 Welcome system for new members
* 📜 Community rules embed
* ⚙️ Modular command system using **Cogs**
* 🧹 Basic moderation commands
* 💬 Utility commands for members

More features will be added soon.

---

## 🛠️ Technologies

* **Python**
* **discord.py**
* **Git & GitHub**

## 🚀 Setup

Clone the repository:

```
git clone https://github.com/SEUUSUARIO/Misa_Bot
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the bot:

```
python bot.py
```
